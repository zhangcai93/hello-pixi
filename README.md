小精灵冒险(是 Learn Pixi.js 一书中最后一个案例)
点击屏幕让小精灵会向上飞起来;
如果她撞到柱子上，会弹窗提示`gameOver`, 点击弹窗蒙层可以重新开始游戏;
帮助她通过柱子的间隙到达终点，界面会显示一个巨大 Finish 标志, 会弹窗提示`gameWin`。
